# Next.Js

The fundamental of Next.Js

#### Key Commands

```
- createing a new project
  npx create-next-app@latest   (i will use version 13.4.13)

- Running the dev Server
  npm run dev

- Building the application
  npm run build

- Starting the application in produciton mode
  npm start
```
